# 社区行为准则

本项目讨论亲密关系、性、婚姻、身份、创伤与家庭等敏感主题。参与者必须尊重不同性取向、性别身份、关系结构、文化、年龄、身体状况和生活选择。

## 我们鼓励

- 针对内容、证据和具体行为讨论，而不是攻击个人。
- 在纠错时提供来源、推理或可复现的例子。
- 尊重隐私，使用虚构信息或充分匿名化的案例。
- 接受别人撤回参与、拒绝回答和不同意的权利。
- 在发现危险建议时清楚指出现实伤害和替代方案。

## 不可接受

- 骚扰、威胁、羞辱、歧视或仇恨表达。
- 发布真实姓名、联系方式、私密聊天、照片或其他可识别个人的信息。
- 为暴力、强迫、跟踪、报复、诈骗或绕过同意提供操作指导。
- 将对性别、性取向、人格或群体的刻板印象伪装成科学事实。
- 借项目讨论招募、诱导或骚扰处于脆弱状态的用户。

维护者可以编辑、隐藏或移除违反准则的内容，并限制相关参与者继续互动。安全问题请按 [SECURITY.md](SECURITY.md) 私下报告。
