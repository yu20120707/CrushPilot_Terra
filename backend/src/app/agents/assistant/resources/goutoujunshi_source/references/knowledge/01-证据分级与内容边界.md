# 证据分级与内容边界

## 一、为什么恋爱建议尤其容易失真

亲密关系研究常依赖自我报告，样本又多来自受教育程度较高的西方人群。关系是双人系统，同一句话在不同权力关系、文化、关系阶段和安全环境中可能产生完全不同的效果。此外，能预测“当前满意度”的变量，不一定能预测关系将来会变好或变坏。

2020 年一项汇总 43 个双人纵向数据集、11,196 对伴侣的研究发现，当前关系质量最稳健的关系层面预测因素包括：感知到的伴侣承诺、对伴侣的欣赏、性满意度、感知到的伴侣满意度和冲突；但关系质量随时间的变化仍然很难预测。这意味着应避免承诺“掌握某个指标就能预测婚姻结局”。

## 二、建议采用的证据等级

### A 级：优先使用

- 系统综述、元分析、预注册多研究项目。
- 大样本纵向或双人研究。
- 官方统计、现行法律、权威公共卫生指南。
- 适合表述：“综合证据较一致”“官方定义/现行制度为”。

### B 级：可以使用但要注明限制

- 设计良好的单项实验、自然观察或纵向研究。
- 有理论基础、在多个研究团队中出现的关系过程。
- 适合表述：“一项研究发现”“可能有帮助，仍需结合情境”。

### C 级：用于启发，不作定论

- 小样本、横断面、便利样本、单一文化研究。
- 定性访谈、案例研究、临床经验。
- 适合表述：“可作为提问框架”“不能据此判断因果”。

### D 级：流行框架

- MBTI 配对、星座、爱情语言、网络性别话术、“高价值/低价值”、未经验证的微表情读心等。
- 可以用来打开对话，但不能用于诊断、贴标签或重大决策。

## 三、四个常见推理错误

1. **相关当因果**：满意的人可能更会积极沟通，也可能因为满意才更愿意沟通；两条路径可以同时存在。
2. **平均当个人**：群体平均差异不能自动推断到某个男人、女人或某种人格类型。
3. **标签当解释**：“他回避型所以不沟通”只是改名，没有说明触发情境、具体行为与后果。
4. **技巧脱离权力**：要求被控制或被暴力的一方“更温柔表达”，可能增加危险或把责任推给受害者。

## 四、将抽象概念行为化

不要只问“他爱不爱我”，改问：

- 承诺：是否愿意讨论关系定义、未来安排，且言行长期一致？
- 回应性：我表达需要时，对方是否能理解、尊重并关心，而不是立刻否定或惩罚？
- 尊重：对方能否接受“不”、不同意见、独处和我的其他社会关系？
- 修复：冲突后是否有人重新接触、道歉、承担影响并落实改变？
- 公平：金钱、家务、照护与决策权是否经过协商，双方是否认为合理？

## 五、给未来内容系统的边界

- 不提供操纵、欺骗、跟踪、偷拍、灌醉、隔离、报复或绕过同意的方案。
- 不以性别、性取向、地域、收入、外貌或人格类型贬低某类人。
- 不诊断人格障碍、精神疾病或“依恋类型”；建议专业评估时说明理由。
- 不为暴力关系设计“共同沟通练习”；先做个人安全、法律和专业转介。
- 涉及未成年人、性健康、财产和婚姻法律时，必须提示地区与时效差异。
