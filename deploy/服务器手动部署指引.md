## 用 VS Code 连接后的操作

在 VS Code Remote-SSH 终端中执行：

```bash
cd ~
git clone https://github.com/yu20120707/CrushPilot_Terra.git
cd CrushPilot_Terra/deploy
```

将本地 `deploy/.env` 上传/复制为服务器上的 `deploy/.env`，然后务必将 `POSTGRES_PASSWORD` 改成高强度随机密码。

```bash
docker compose up -d --build
docker compose ps
curl http://127.0.0.1:8080/health
```

浏览器访问：`http://43.138.155.193:8080`。

部署前，在腾讯云安全组放行 TCP `8080`。如果后续绑定域名并启用 HTTPS，放行 TCP `80`、`443`，且反向代理必须对 `/api/v1/chat` 关闭响应缓冲。
